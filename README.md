## üíª Installation  

### Prerequisites  
- macOS 12.0 (Monterey) or later  
- An API key from a supported AI provider (e.g., OpenAI, Google AI)  

### Steps  
1. **Download** ‚Äì Grab the latest release of `AI Tab Bar.app` from the [Releases Page](./releases).  
2. **Move to Applications** ‚Äì Drag the app into your `/Applications` folder.  
3. **Launch** ‚Äì Double-click the app icon to start.  
4. **Initial Setup** ‚Äì Enter your API Key and choose your preferred AI provider/model on first launch.

## üöÄ Usage  

1. **Activate** ‚Äì Press your global shortcut (`Cmd + Shift + A` by default).  
2. **Type** ‚Äì Enter your query in the search bar:  
   - `What is the capital of New Zealand?`  
   - `Write a short summary of the plot of Dune in three sentences.`  
   - `Explain this Python code snippet: [i for i in range(10)]`  
3. **Receive Answer** ‚Äì AI streams the response directly below your input.  
4. **Interact** ‚Äì Copy, paste, or continue the conversation.  
5. **Dismiss** ‚Äì Press `Esc` or click outside the window.

## ‚öôÔ∏è Configuration  

Open the **settings panel** via the **gear icon** in the Tab Bar or from the **macOS menubar**.  

| Setting              | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| **Global Shortcut**   | Customize the key combination to open the Tab Bar.                         |
| **API Key**           | Required ‚Äì Enter your API key for the chosen AI provider.                  |
| **AI Provider**       | Select your backend service (OpenAI, Gemini, Anthropic, etc.).             |
| **AI Model**          | Choose the model (e.g., GPT-4o, Gemini 1.5 Pro).                           |
| **Max History Length**| Set the number of previous queries and conversations to store.             |
| **Launch at Login**   | Enable auto-launch when macOS starts.                                      |
