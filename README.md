## 🚀 Overview

**AI Tab Bar** is a macOS utility that replaces or complements the built-in **Spotlight Search**, letting you ask questions, summarize content, explain code, translate text, and perform tasks — all through AI.  
Open it from **anywhere** with a global shortcut and get fast, contextual responses in a small, elegant popup.

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| ⚡ **Global Shortcut** | Press `Cmd + Shift + A` (default) to open from any app. |
| 🧠 **Instant AI Responses** | Get concise, smart answers powered by modern LLMs. |
| 🔄 **Context-Aware Queries** | Summaries, translations, explanations, creative writing & more. |
| 🪶 **Clean macOS UI** | Minimal Spotlight-like popup that hides when dismissed. |
| 🔌 **Customizable AI Provider** | Choose between OpenAI, Gemini, Claude, etc. |
| 📝 **Search History** | Automatically saves previous conversations. |

## 💻 Installation

### Requirements
- macOS **12.0+**
- API Key from any supported AI service (OpenAI, Gemini, Anthropic, etc.)

### Steps

1. **Download** the latest `.app` from the **[Releases Page](./releases)**  
2. Move it into your **`/Applications`** folder  
3. Open the app  
4. Enter your **API Key** and select provider/model  

---

## 🎯 Usage

1. Press **`Cmd + Shift + A`**
2. Type your query:
   - `What is the capital of New Zealand?`
   - `Explain this JS function: () => console.log("Hi")`
   - `Summarize Dune in 3 sentences`
3. View the answer instantly
4. Copy, continue, or dismiss (`Esc`)

---

## ⚙️ Settings

| Setting | Description |
|--------|-------------|
| ⌨️ **Global Shortcut** | Customize your activation keys |
| 🔑 **API Key** | Required for AI access |
| 🤖 **AI Provider / Model** | Switch between backends (GPT-4o, Gemini 1.5 Pro, Claude, etc.) |
| 🗂 **History Length** | Limit stored past queries |
| 🚀 **Launch at Login** | Open on macOS startup |
